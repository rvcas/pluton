# pluton

A desktop app for Cardano devs built with [iced](https://iced.rs).

### Features

- [ ] decode and display a block
- [ ] decode and display a tx
- [ ] decode and display an address
- [ ] resolve inputs
- [ ] watch chain sync live
- [ ] watch mempool live
- [ ] uplc debugging and aiken integrations

### Dev

```sh
cargo watch -x run
```
